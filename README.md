Barbie Movie Catalog

Catálogo de filmes da Barbie desenvolvido para praticar desenvolvimento web, organização de conteúdo e criação de interfaces temáticas inspiradas no universo Barbie.

Funcionalidades:
Listagem de filmes da Barbie
Exibição de capa, título e descrição
Interface temática personalizada
Navegação simples e intuitiva
Organização visual inspirada no estilo Barbie
Tecnologias utilizadas
HTML
CSS
JavaScript


O projeto foi desenvolvido com o objetivo de praticar:
Estruturação de páginas web
Estilização com CSS
Organização de catálogos
Manipulação de elementos visuais
Experiência do usuário (UI)


Durante o desenvolvimento deste projeto, foram praticados conceitos de:
Desenvolvimento Front-End
Organização visual de conteúdo
Estruturação de componentes
Design temático
Responsividade básica
